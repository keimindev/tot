
# T.O.T (Trace of Time)

### ✍🏻Summary
매일 30분씩 책 읽기를 하는 중인데 기존에 사용중인 북어플로 시간을 측정하면 한 달 혹은 일주일 독서 시간을 확인하기 힘들고 원서를 주로 읽어야하는 불편함이 있어서 나만의 기록장을 만들기로 함.
<br/>
<br/>

### ⚡️key features
- 스톱워치
- 웗 별 활동 총량 기록
- 섹션별 시간 기록
- 로그인 (github계정)
<br/>
<br/>
